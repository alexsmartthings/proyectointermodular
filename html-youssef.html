2.1.3.2 Instalación de netbeans :

- Visita el sitio oficial de netbean :

- Cuando haga clic en descargar, el sitio le brindará las diferentes versiones del programa :

- Elija una versión (depende de su dispositivo y necesidades) y haga clic en descargar :

- Espere hasta que esté terminado

- Cuando haya terminado, deberá completar la configuración del programa haga clic en next :

- Aceptar y hacer clic en next :

- Aquí te muestra la carpeta en la que quieres instalar Apache :

- Cuando termines haz clic en instalar :

- New Project: Crea un proyecto nuevo en lenguajes como Java o PHP.
- Open Project: Abre un proyecto existente.
- New File: Genera un archivo nuevo, como clases o interfaces.
- Open File: Abre archivos existentes en el proyecto.
- Go to File: Busca rápidamente archivos dentro del proyecto.
- Show Dashboard: Muestra un panel con información sobre tus proyectos.

- El kit de desarrollo de Java (JDK) es esencial para ejecutar y desarrollar proyectos Apache basados ​​en Java, como Tomcat, Kafka y Hadoop. Proporciona herramientas como el compilador de Java, el entorno de ejecución (JRE) y las bibliotecas. Muchas herramientas de Apache requieren el JDK para compilar, crear o ejecutar aplicaciones. Asegúrese de que el JDK esté instalado, que la variable JAVA_HOME esté configurada y que la versión correcta coincida con los requisitos de su proyecto para un funcionamiento sin problemas.

2.1.3.3 Instalación de intellij IDEA :

- Visita el sitio oficial de intellij idea y haga clic en download (descargar) :

- Haga clic en instalar y espere :

- Espera hasta que termine :

- Puedes elegir el destino de tu carpeta cuando lo hayas hecho, haz clic en next :

Elige una de las opciones depende de ti y tus necesidades :
    - Acceso directo en el escritorio: Crea un ícono en el escritorio para abrir IntelliJ rápidamente.
    - Actualizar PATH: Permite ejecutar IntelliJ desde la terminal con comandos como idea.
    - Menú contextual: Agrega la opción "Abrir carpeta como proyecto" al hacer clic derecho en carpetas.
    - Asociaciones de archivos: Abre automáticamente archivos como .java, .kt y .gradle con IntelliJ.

- Cuando termines haz clic en next :

- Aquí puedes elegir la carpeta donde quieres colocar el acceso directo del programa elige una de las opciones a continuación Cuando termines haz clic en install :

- Projects: Permite abrir un proyecto existente o crear uno nuevo directamente desde la interfaz.
- Remote Development: Facilita la conexión con entornos de desarrollo remotos para trabajar sin importar dónde se encuentre el código.
      SSH: Conecta con máquinas remotas a través de SSH para trabajar de manera segura.
      WSL: Integra el Subsistema de Windows para Linux (WSL) como un entorno de desarrollo.
- Dev Containers: Permite trabajar dentro de contenedores diseñados específicamente para desarrollo.
- Customize: Ofrece opciones para ajustar la configuración y la apariencia de IntelliJ IDEA según tus preferencias.
- Plugins: Administra, instala o actualiza los plugins que expanden las capacidades del IDE.
- Learn: Accede a tutoriales, documentación y recursos de aprendizaje para dominar IntelliJ IDEA fácilmente.

2.1.3.4 Instalación de Eclipse :

- Visita el sitio oficial de intellij idea y haga clic en download (descargar) :

Haga clic en instalar.
- Puede editar antes de la instalación según sus necesidades :

- leer y accepter :

  - Review IDE Configuration Settings : Personalice las preferencias del IDE, como temas, estilos de código y opciones del compilador.
  - Create a Hello World Application : Siga un tutorial guiado para crear y ejecutar un programa de "Hola mundo" simple en Java.
  - Create a New Java Project : Configure rápidamente un nuevo proyecto de Java desde cero con un asistente.
  - Checkout Projects from Git : Clone e importe proyectos desde un repositorio de Git a su espacio de trabajo.
  - Import Existing Projects : Importe proyectos de Eclipse o externos creados previamente a su espacio de trabajo.
  - Launch the Eclipse Marketplace : Acceda e instale complementos o extensiones para mejorar la funcionalidad del IDE.
  - Open an Existing File : Abra directamente cualquier archivo desde su sistema de archivos en Eclipse.
